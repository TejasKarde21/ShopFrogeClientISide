# ShopForge Admin

A client-side admin dashboard demo for ShopForge, built as a single-page HTML application with React and in-browser data management.

## Features

- Admin dashboard mockup for product, category, and order management
- In-memory data store for users, products, orders, payments, and store info
- Dark UI with realtime table and card views
- Demo login page and dashboard components rendered with React

## Run

Open `index.html` in your browser.
